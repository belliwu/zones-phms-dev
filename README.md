# zones-phms-dev

> Assisted Reproductive Management System for TMUH
---
## Devlopement Setup
```bash
# Initial vue webpack zones-phms-dev
vue init webpack zones-phms-dev

# The following CSS assets are required:
in index.html :
 1. /css/bootstrap.min.css
 2. /css/bootadmin.min.css
 3. /css/all.css

Below are option
 4. /css/datatables.min.css
 5. /css/fullcalendar.min.css

# The following JS assets are required:
 1. /js/jquery.min.js
 2. /js/bootstrap.bundle.min.js
 3. /js/bootadmin.min.js

  Below are option
 4. /js/datatables.min.js
 5. /js/moment.min.js
 6. /js/fullcalendar.min.js
```

## Build Setup
```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
